# Quick Vercel Deployment Guide

## Prerequisites
- GitHub account
- Vercel account (free at vercel.com)

## Step-by-Step Deployment

### 1. Prepare Your Repository

```bash
# Initialize git if not already done
git init

# Add all files
git add .

# Commit
git commit -m "Initial commit: Integration Coach app"

# Create GitHub repo (via GitHub website or CLI)
# Then add remote and push
git remote add origin https://github.com/YOUR_USERNAME/YOUR_REPO_NAME.git
git branch -M main
git push -u origin main
```

### 2. Deploy to Vercel

**Option A: Via Vercel Dashboard (Easiest)**

1. Go to [vercel.com/new](https://vercel.com/new)
2. Click "Import Project"
3. Select "Import Git Repository"
4. Authorize Vercel to access your GitHub
5. Select your repository
6. Vercel will auto-detect configuration:
   - **Framework Preset:** Vite
   - **Build Command:** `npm run build`
   - **Output Directory:** `dist`
   - **Install Command:** `npm install`
7. Click "Deploy"
8. Wait 2-3 minutes
9. Copy your deployment URL (e.g., `https://your-app-name.vercel.app`)

**Option B: Via Vercel CLI**

```bash
# Install Vercel CLI
npm install -g vercel

# Login to Vercel
vercel login

# Deploy (run from project root)
vercel

# Follow prompts:
# - Link to existing project? No
# - Project name? [Enter your desired name]
# - Directory? ./
# - Override settings? No

# Deploy to production
vercel --prod
```

### 3. Get Your URL

After deployment, Vercel provides:
- **Production URL:** `https://your-app-name.vercel.app`
- **Preview URLs:** Auto-generated for each commit

Copy your production URL - this is what you'll embed in Framer!

### 4. Embed in Framer

Use this iframe code in Framer:

```html
<iframe 
  src="https://your-app-name.vercel.app" 
  style="width: 100%; height: 100vh; border: none;"
  title="Integration Coach"
  allow="clipboard-read; clipboard-write"
></iframe>
```

## Troubleshooting

### Build Fails

**Check package.json scripts:**
```json
{
  "scripts": {
    "dev": "vite",
    "build": "vite build",
    "preview": "vite preview"
  }
}
```

**Common fixes:**
```bash
# Clear node_modules and reinstall
rm -rf node_modules package-lock.json
npm install

# Test build locally
npm run build

# If successful, commit and push
git add .
git commit -m "Fix build configuration"
git push
```

### App Doesn't Load

1. Check Vercel deployment logs for errors
2. Ensure `vercel.json` has proper rewrites configuration
3. Verify all dependencies are in `package.json` (not just devDependencies)

### Clipboard Not Working in Framer Iframe

Add to iframe:
```html
allow="clipboard-read; clipboard-write"
```

### App Works Locally but Not on Vercel

- Check for environment-specific code
- Ensure all imports use correct paths (case-sensitive on Vercel)
- Verify no absolute URLs pointing to localhost
